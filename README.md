<h1 align=center>Реализация шаблона CRUD </h1>

<h2>Цель: Разработать и реализовать клиент-серверную информационную систему, реализующую механизм CRUD.</h2>

<h2>Задание</h2>

Система предназначена для анонимного общения в сети интернет.
Интерфейс системы представляет собой веб-страницу с лентой заметок, отсортированных в обратном хронологическом порядке и форму добавления новой заметки.

Возможности пользователей:

- добавление заметки (Create, Read)
- реагирование на чужие заметки (лайки и дизлайки)
- удаление заметки (Delete)
- редактирование заметки (Update)

В лабораторной работе используется bootstrap.

<h2 align=center>5. Описание алгоритмов</h2>

1. Создание заметки: пользователь нажал соответствующую кнопку, открылась форма, пользователь ввёл имя и сообщение и подтвердил создание / отменил создание записи. В случае подтверждения, предоставленные пользователем данные заносятся в БД.
2. Редактирование заметки: пользователь нажал соответствующую кнопку, открылась форма, пользователь отредактировал имя и сообщение и подтвердил изменения / отменил оные. В случае подтверждения, соответствующая запись в БД изменяется.
3. Удаление заметки: пользователь нажал соответствующую кнопку, открылась форма, пользователь подтвердил или отменил удаление заметки. В первом случае, соответствующая запись удаляется из БД.
4. Реагирование на заметки: пользователь нажал соответствующие лайку и/или дизлайку кнопки, значение лайков/дизлайков соответствующей заметки увеличилось на единицу, произошло обновление соответствующей заметке записи в БД.
